# AI Index — 2026

- 16개 카테고리, 48개 도구
- 사용자 수 · 평판 · 전문가 평가 종합 랭킹
- 카드 클릭 → 카테고리 상세 비교 / 1·2·3위 행 클릭 → 공식 사이트
